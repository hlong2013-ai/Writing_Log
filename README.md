# The Writing Log

A simple personal writing tracker I created for my own use.

I’m not a professional software developer or “real coder” by trade. This project was built as a practical tool to help me track writing sessions, manuscript progress, journal notes, and project goals in a way that works for my own writing process.

That said, anyone is welcome to look through the code, use it for their own personal writing tracker, adapt it, or improve it.

## What it does

The Writing Log is a single-file HTML app that runs in your browser. It can help track:

- Daily writing sessions
- Word counts
- Writing time
- Manuscript progress
- Project deadlines and goals
- Journal notes and writing reflections
- Multiple writing projects
- Completed drafts

## How to use it

Open the HTML file in your browser and start tracking your writing.

Your data is stored locally in your browser using `localStorage`. You can also export your data as a JSON file and import it again later.

## Privacy note

This tool does not require an account, server, database, or backend.

Your writing data stays in your own browser unless you choose to export it. Exported JSON files may contain personal project titles, writing logs, journal entries, and progress data, so be careful not to upload those files to a public GitHub repository unless you intentionally want to share them.

Only import JSON files you trust.

## Why I made this

I wanted a writing tracker that felt more personal and motivating than a spreadsheet, while still being simple enough to run as a single local file. It was made for my own writing process, but I’m sharing it in case it is useful to someone else.
